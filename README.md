# breast-cancer-detection-using-NN

##  Overview

This project predicts whether a tumor is **malignant** or **benign** using NEURAL NETWORK.
It is based on the Breast Cancer Wisconsin dataset.

---

##  Dataset

* Breast Cancer Wisconsin Dataset
* Features include radius, texture, perimeter, area, etc.

---

##  Workflow

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Model training
* Model evaluation

---

##  Models Used

* NEURAL NETWORK

---

##  Results

* Accuracy: 95.61%
* Model performed well in classifying tumors
